Aprendiendo a usar HTML!

En este documento veremos distintas funcionalidades que HTML nos brinda para gestión semantica de los textos. En particular analizaremos dos funcionalidades denominadas "em", "strong" y "b".
Si bien podemos asociar dichas herramientas a una cuestión estetica, en las proximas secciones, veremos que no es así.

Uso de "em"

En este caso "em" nos permite hacer una diferenciación del texto que si bien brinda un nueva estetica no es la unica función ya que en funcionalidades de accesibilidad brinda una diferenciación, por ejemplo, en el tono de la lectura de dicho texto.
Esta sección esta destacada semanticamente con el uso de "em".

Uso de "strong"

En este caso "strong" Tambien nos permite hacer una diferenciación del texto desde un punto de vista semantico, agnostico del soporte de acceso.
Esta sección esta destacada semanticamente con el uso de strong.

Uso de "b"

En este caso "b" también nos brinda funcionalidades de distinción semantica, con la diferencia de que esta funcionalidad se está dejando de utilizar.
Esta sección esta destacada semanticamente con el uso de "b".
